# Contributing to akaire

Thank you for your interest in contributing!

## DCO (Developer Certificate of Origin)

All commits must be signed off with `git commit -s`, adding a line:

```
Signed-off-by: Your Name <your.email@example.com>
```

By signing off, you certify the [DCO](https://developercertificate.org/).

## 現状のフェーズ

`akaire` は pre-MVP 段階です。コード貢献の前に、Issue で設計議論に参加してください。

- 設計 ADR: `docs/adr/`
- 技術 RFC: Issues の `label:rfc`
- バグ報告・機能要望: 通常の Issue

## Pull Request

1. Fork して branch を切る
2. 変更を小さく保つ（1 PR = 1 concern）
3. Commit に `git commit -s` で署名
4. PR テンプレに従って記述
5. CI が通るまで修正

## ライセンス

貢献いただいたコードは Apache-2.0 の下で公開されることに同意いただけたものとします。
